# Sales Data Analysis

## Project Overview

This project analyzes sales data using the Pandas library in Python. It demonstrates how to load a CSV file, explore the dataset, clean the data, calculate business metrics, and generate a simple sales report.

## Objectives

* Load CSV dataset
* Explore dataset
* Handle missing values
* Remove duplicate records
* Calculate sales metrics
* Generate sales report

## Technologies Used

* Python
* Pandas
* VS Code
* GitHub

## How to Run

Install Pandas

pip install pandas

Run the project

python sales_analysis.py

* Total Revenue
* Average Sale
* Highest Sale
* Lowest Sale
* Best Selling Product
* Revenue by Region
* Revenue by Product
