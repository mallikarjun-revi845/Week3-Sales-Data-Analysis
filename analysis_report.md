# Sales Data Analysis Report

## Project Description

The objective of this project is to analyze sales data using Python and the Pandas library. The project loads the dataset, performs data cleaning, calculates business metrics, and generates a sales report.

## Analysis Steps

1. Load Dataset
2. Explore Dataset
3. Check Missing Values
4. Remove Duplicate Records
5. Calculate Total Revenue
6. Find Highest & Lowest Sales
7. Find Best Selling Product
8. Analyze Revenue by Region
9. Generate Sales Report

## Findings

* Dataset Loaded Successfully
* Missing Values Checked
* Duplicate Records Removed
* Total Revenue Calculated
* Best Selling Product Identified
* Region-wise Revenue Calculated

## Conclusion

This project demonstrates how Python and Pandas can be used for basic business data analysis and reporting.
